Configuration
dropDownWidth : String
The width for the Kendo DropDownList widget. This is optional. If it is not provided, the Kendo DropDownList will use the default widget.
treeview : kendo.ui.TreeView
The configuration settings for the Kendo TreeView.
Methods
dropDownList
Returns a reference to the Kendo DropDownList.
treeview
Returns a reference to the Kendo TreeView.
Events
select
Triggered when a node is being selected by the user.
