Alert Dialog : kendo.ui.ExtAlertDialog

Configuration
message : String
Message to be displayed to the user.
icon : String
The CSS Class of the image to be displayed. Images provided are:
  • Information: k-ext-information
  • Question: k-ext-question
  • Warning: k-ext-warning
  • Error: k-ext-error
Methods
show
Show the alert dialog.
Parameters
options : JSON Object
Configuration options used to display the alert dialog.
Return
jQuery.Deferred Object
jQuery Deferred object. When the Deferred object is resolved, any done callbacks are executed when the user clicks on the OK button.
hide
Hide the alert dialog.

OK/Cancel Dialog : kendo.ui.ExtOkCancelDialog

Configuration
message : String
Message to be displayed to the user.
icon : String
The CSS Class of the image to be displayed. Images provided are:
  • Information: k-ext-information
  • Question: k-ext-question
  • Warning: k-ext-warning
  • Error: k-ext-error
Methods
show
Show the ok/cancel dialog.
Parameters
options : JSON Object
Configuration options used to display the ok/cancel dialog.
Return
jQuery.Deferred Object
jQuery Deferred object. When the Deferred object is resolved, any done callbacks are executed when the user clicks on the OK or Cancel button. The argument returned is a JSON Object with the following attributes:
  • button (String): the button that was clicked.
  • input (String): the input entered by the user.
hide
Hide the ok/cancel dialog.
